How to use the force?

Installing the Force

To use the Scroll Force, include the scroll-force.js file in your HTML code.

<script src="scroll-force.js"></script>

For direct use,

<script src="https://amaljose96.github.io/scroll-force/scroll-force.js"></script>

Using the Force

To use the force, you need to add rules or triggers.

addrule(<item>,<type>,<property>,<start_value>,<end_value>,<unit>,<check_type>,<check_start>,<check_end>);

Item

This field is filled with the entity to which the force is to be applied to. A jquery identifier is used here.

Type

As of the current version, there are 3 types.

  • Color:This is used incase the force you want to apply is on a color.eg. background-color.
  • Style:This is used for changing a value of CSS attribute. eg. padding, margins etc.
  • Attribute:This is used for changing an atrribute of that item

    • Property

    This refers to the CSS attribute to be changed. For current version, except for colors, only numerical values are supported.

    Start Value,End value and Unit

    The start values, end values and unit for the property.

  • Color: This can either be a hex value or rgb. eg. "#deffce" or "#eff" or {r:100,g:200,b:124}. For Using hex values, the unit is "hex". For {r,g,b}, unit is "rgb". For {r,g,b,a}. the unit is "rgba". 
    addrule("body","color","background-color",{r:240,g:240,b:240},{r:250,g:240,b:240},"rgb",'value',0,100);
  • Style: The start and end numbers are the start and end values. Unit can be "px", "em" or any other unit. If the property doesnt have a unit (eg. opacity), leave it as "". 
    addrule("#element","style","margin-top",10,100,"px",'value',0,100);
  • Attribute: This is also a number for start and end values. Units can be "px" or any other valid unit. 
    addrule("#element","attribute","height",10,100,"px",'value',0,100);

    Check type, check start and check end.

    There are 2 check types - percent and value. Value corresponds to the position of the scrollbar in pixels on the screen. Percent corresponds to percentage scrolled on the screen.

    Triggers

    addtrigger(<function>,<direction>,<check_type>,<check_point>);
    Eg. 
     addtrigger(function(){$("#text").html("And night");},"down","value",900);

    Function contains code that has to be executed at check point.

    Direction ("up"/"down") specifies if we it is to be triggered when we scroll up or down

    Check type and Check point are same as rule check. Type can be percent or value.
